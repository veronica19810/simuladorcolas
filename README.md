# Simulador de Sistema de Colas 

Este proyecto simula una **fila (cola)** como en un supermercado, donde las personas llegan, esperan y son atendidas.

## ¿Qué hace?
- Simula llegadas y servicios aleatorios
- Calcula el tiempo de espera y atención
- Guarda los resultados en un archivo CSV
